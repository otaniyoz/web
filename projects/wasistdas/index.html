Wasistdas: language practice with a pop

This is a simple puzzle for practicing basic language vocabulary:

The playing mechanics was inspired by popping bubble wrap - you play the game by popping the icon that matches the word in one of the selected languages.

Correct taps come with a rewarding pop, incorrect - with a silent shake.

Tap to unmute. Tap again to mute.

The dark colour scheme support is realised through pure CSS using a media query and a single-line filter:
